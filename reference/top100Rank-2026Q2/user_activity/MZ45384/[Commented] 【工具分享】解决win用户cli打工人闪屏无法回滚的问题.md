## 帖子正文

经常看到顾问村和mcp交流群里有顾问说gemini cli、iflow cli或者打工人总是闪屏，并且在跑工作流时不能进行回滚，同时也会有回滚时显示的历史对话不全等等的问题。这些问题并不是小问题，查看历史对话可以清楚知道AI都干了什么，也方便管理你多开的工作流。

### 解决办法

更换win自带的cmd工具，下载链接地址： [https://apps.microsoft.com/detail/9n0dx20hk701?hl=zh-CN&gl=US](https://apps.microsoft.com/detail/9n0dx20hk701?hl=zh-CN&gl=US)   ![图片](images/img_8ce63d42c6.png)

### 注意

1. 下载完如果找不到的话，左下角搜索终端
2. powershell进不去cli的话就用cmd进去
3. 仅限windows用户
