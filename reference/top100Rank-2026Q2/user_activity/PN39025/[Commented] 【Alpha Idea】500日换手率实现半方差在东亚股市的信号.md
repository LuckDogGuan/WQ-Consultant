## 帖子正文

本alpha灵感基于以下两篇文章:

> [Good Volatility, Bad Volatility and the Cross-Section of Stock Returns by Tim Bollerslev, Sophia Zhengzi Li, Bingzhi Zhao :: SSRN](https://papers.ssrn.com/sol3/papers.cfm?abstract_id=2565660)
> 【中信建投】量化策略：中高频波动率在商品期货里的影响

主信号：实现半方差

> 投资者在用业绩波动去衡量基金风险的同时，潜意识里早已为波动加上了“好”与“坏”的标签。向上波动代表基金赚钱，是“好的”波动，向下波动代表基金赔钱，这是“坏的”波动。所以很多投资人倾向于认为，向上波动是机会，向下波动才是风险。

使用实现半方差去衡量波动的好与坏，做多正半方差的股票，做空负半方差的股票。

对于股票/期货日内的高频数据而言，实现半方差是个动量alpha，然而根据  [XD81759](/hc/en-us/profiles/23494746482967-XD81759)

[【Alpha灵感】Template：半方差 && ChatGPT在量化研究中的一些应用 – WorldQuant BRAIN](/hc/en-us/community/posts/25275381972247--Alpha%E7%81%B5%E6%84%9F-Template-%E5%8D%8A%E6%96%B9%E5%B7%AE-ChatGPT%E5%9C%A8%E9%87%8F%E5%8C%96%E7%A0%94%E7%A9%B6%E4%B8%AD%E7%9A%84%E4%B8%80%E4%BA%9B%E5%BA%94%E7%94%A8)

的研究，在GLB市场下，日频化的实现半方差往往呈现出反转的信号。

基于以上内容，本文使用换手率构建实现半方差alpha，并在CHN,KOR,JPN(TOP1600,TOP1200),ASI(MINVOL1M)四个region；五个universe下测试其效果。

核心表达式：

![图片](images/img_dd642657c5.png)

```
turn = volume/(sharesout*1000000);turn_up = ts_sum(power(turn,2)*if_else(turn>0,1,0),500);turn_down = ts_sum(power(turn,2)*if_else(turn <0,1,0),500);
```

其中，500来自于中证期货，使用500根K线（日内）计算因子，本文则用500根日线计算。

在使用换手率的环境下，正半方差表示的是放量的波动，而负半方差表示的是缩量的波动。

**总结与探索：**

1. 与前面的同学在glb上面的研究结果一致，实现半方差（不管是换手率还是收益）构造的alpha在日频总体上是反转因子。推动缩量/收益下降的“压力波动”会给股票在未来得到超额收益，而“推力波动”则会使得股票在未来下跌。

2. 在CHN\KOR\JPN\ASI 四个region，500日换手率的实现半方差都有明显的信号。但是在CHN还不够强，过不了sharpe的测试（此外CHN的prod correlation较高，说明有人交过类似的）。

3. 该alpha在CHN，JPN， KOR体现了高强度的一致性，就是收益大部分来自于小市值的股票

![图片](images/img_554e426ed8.png)

结合这一点探讨500日的半方差呈现的反转效应的可能原因：日频投资者的非理性行为，俗称的“炒作”（体现在波动率高这一点上），会导致价格在未来反转。

4.然而，在ASI，这个策略的收益却更多体现在中大市值的股票上。

![图片](images/img_2c3e7aed21.png)

这里猜测是构建alpha的时候没有对冲掉市值里面受国家影响的部分，因为该策略即使对冲掉了country，其收益也主要来自于日韩两个在亚洲市值相对大的国家。
