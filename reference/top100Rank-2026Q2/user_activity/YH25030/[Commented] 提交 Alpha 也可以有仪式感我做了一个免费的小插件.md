## 帖子正文

做因子、跑回测、等提交结果……

相信很多人都和我一样：流程很熟，但情绪波动也很真实。

有时候一个 Alpha 调了很久，点下 Submit 后只能干等；
有时候切到别的页面，回来才发现结果早就出来了。
所以我做了一个小工具，目标很简单： **让提交这件事更清晰、更有反馈，也更顺手。**

## 这个插件能做什么？

### 1）提交成功/失败，有明确反馈

- ✅ 成功：会有庆祝动画（烟花），给自己一点“过关”的正反馈
- ❌ 失败：不会放庆祝动画，但会有轻反馈 + 明确提示，不会突然“没反应”

![图片](images/img_5441f6ab22.jpeg)

![图片](images/img_56223ce852.jpeg)

### 2）等待结果时，有“正在等待”的状态

提交后有时接口返回会慢，尤其在网络波动或高峰时段。
插件会显示“等待中”状态，避免你反复猜测“到底提交到了没有”。

![图片](images/img_03bcdfd6da.png)

### 3）切走页面也不怕错过结果

很多人提交通常不会盯着同一页一直等。
插件支持你切到其他页面继续工作，等结果出来后还能快速回到对应页面查看反馈。

### 4）一键跳转到 Alpha 页面

支持输入 Alpha ID 或相关Location链接，快速打开对应 Alpha 页面。
对于频繁查结果、回看记录的人很省时间。

![图片](images/img_3de1b89946.png)

## 适合哪些人？

- 经常提交 Alpha、希望减少“等待焦虑”的同学
- 喜欢更清晰反馈、减少重复点击/重复确认的人
- 想把流程做得更顺手一点的研究者

## 为什么我愿意免费分享？

这个插件最初就是我自己在日常使用中遇到痛点后做的。
后来发现很多同学也有类似体验，所以决定公开免费分享。
如果它能帮你节省一点时间、减少一点心累，就值了。

## 使用方式（本地安装，免费）

1. 进入 [下载链接](https://drive.google.com/drive/folders/1bouPGQL2_dHyGR1Hndysj1n86qOaUUuI) ，将压缩包解压文件夹
2. 打开 Chrome 扩展管理页 chrome://extensions
3. 开启右上角「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择插件目录（文件夹）即可

![图片](images/img_591fc54a35.png)

## 如何更新

点下插件弹窗右下角检查更新按钮，我把插件放到了谷歌网盘中，下载最新版本zip压缩包安装即可（随机更新~

![图片](images/img_4f6ff71f32.png)

## 最后

这个工具不是“提高收益率”的魔法，
它做的是把你每天都在重复的那一小段流程，变得更流畅一点。
