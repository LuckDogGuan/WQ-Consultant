# 临时解决alpha页面不能翻页的问题

- **链接**: [Commented] 临时解决alpha页面不能翻页的问题.md
- **作者**: WW15616
- **发布时间/热度**: 3个月前, 得票: 2

## 帖子正文

打开开发者工具(快捷键f12),切换到控制台(Console).输入以下命令即可翻页:

```
document.querySelector('.footer__button.footer__button--next').click()
```
