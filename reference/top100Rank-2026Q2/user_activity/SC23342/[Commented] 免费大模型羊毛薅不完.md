## 帖子正文

大家最近都在用LLM，想必各种模型都尝试了七七八八，但对于兼职的我们来说，性价比是必须考量的因素，虽然免费的不如收费的好，但有很多的免费或许能抵得过单个收费。最近发现一个网站可以免费薅到小米MiMio，登录ZenMux，网站，网址 [https://zenmux.ai/，点击网站上方对应的模型，如图：](https://zenmux.ai/%EF%BC%8C%E7%82%B9%E5%87%BB%E7%BD%91%E7%AB%99%E4%B8%8A%E6%96%B9%E5%AF%B9%E5%BA%94%E7%9A%84%E6%A8%A1%E5%9E%8B%EF%BC%8C%E5%A6%82%E5%9B%BE%EF%BC%9A)

![图片](images/img_d10f38cad7.png)

进入后下拉网页到这里：

![图片](images/img_636eefe739.png)

点击Creat API Key，这里需要注册账号，点击注册，可以用Google邮箱登录，然后会弹出如下页面：

![图片](images/img_1f98307a9a.png)

看到这里，具有经验的顾问们应该已经很熟悉了，右边CreateAPIKey按钮先生成自己的API，Base URL是对应的免费模型地址，V1就是小米MiMo，另外Models标签里进入可以选择其他模型。如图：

![图片](images/img_d700da6065.png)

有了URL和API以后就可以到我们熟悉的app里去七十二变或者找灵感了。

另外，欢迎大家将自己找到的免费大模型资源在评论区共享，将我们的WQ事业性价比最大化。
